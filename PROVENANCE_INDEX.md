## HOW TO USE THIS INDEX

**To verify a specific claim:** Find it in the table below. The "Source" column
gives the directory and filename in `CODEX_AURA_PRIME/`. The "PDF Page" column
gives the approximate page range in the primary corpus PDF
(`A SOVEREIGN SYSTEM FOR HUMAN–AI CO-CREATION-merged.pdf`).

**To audit a framework:** Filter to the framework's section. All load-bearing
claims for that framework are listed with provenance.

**PDF page estimates:** The page numbers are estimated from the document structure.
The primary corpus PDF was assembled from five successive versions; section headers
provide anchors for navigation. Where a claim first appeared in an early version and
was refined in later versions, the page given is where the most current version appears.

## PDF PAGE ESTIMATION METHODOLOGY

The primary corpus PDF (`A SOVEREIGN SYSTEM FOR HUMAN–AI CO-CREATION-merged.pdf`,
1,402 pages) was assembled from five versions:
- Version 1: pages 1–280 (approximately)
- Version 2: pages 281–560
- Version 3: pages 561–840
- Version 4: pages 841–1,120
- Version 5: pages 1,121–1,402

Most framework essentials files were developed in Version 3–4 (pages 561–1,120).
Mathematical foundations were developed in Version 4–5.

Page numbers in this index are estimated from section headers and structural position
in the merged PDF. For precise location: search the PDF for the claim text or the
theorem number — each theorem is labeled with its identifier (C1, T1, L1, etc.) in
the original document.
