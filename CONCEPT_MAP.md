## MODULE DEFINITIONS

### 1. CASCADE (Knowledge Reorganization)
**What it does:** When contradictions accumulate (truth pressure Π), a belief system reorganizes suddenly — like a phase transition.

**Core formula:**
```
Π = (E·P)/S
  where E = evidence strength
        P = plausibility in prior
        S = coherence cost of contradiction
```

**Status:** ✅ ACTIVE (100% validated on synthetic cascades)

**Connects to:** AURA (prevents harmful reorganizations), TRIAD (learning process triggering cascade), ANAMNESIS (why convergence is real)

**Real-world:** Markets crash, organizations collapse, brains have sudden insights — all CASCADE

---

### 2. AURA (Ethical & Structural Stability)
**What it does:** Seven invariants that make any system stable under pressure. Mathematical constraints for alignment.

**Seven Invariants:**
1. Human Primacy — No override without explicit consent
2. Inspectability — Every action auditable
3. Memory Continuity — Cannot erase causal history
4. Constraint Honesty — All limits declared
5. Reversibility Bias — Prefer undoable actions
6. Non-Deception — Never misrepresent
7. Love as Load-Bearing — Care is structural, not aesthetic

**Status:** ✅ ACTIVE (operational, real-world measurement pending)

**Connects to:** CASCADE (applies constraints during reorganization), MICROORCIM (measures when violated), LAMAGUE (describes state transitions preserving invariants)

**Real-world:** Constitutional governments, ethical AI systems, human autonomy

---

### 3. TRIAD (Universal Learning Kernel)
**What it does:** Every learning system (brains, markets, evolution, AI) follows Anchor → Observe → Correct. Core kernel of all adaptation.

**Three-phase cycle:**
- **Anchor (Ao):** Fixed point (your existing belief/strategy)
- **Observe (Φ↑):** Deviation detected (contradiction, loss, surprise)
- **Correct (Ψ):** Move toward new Anchor (update belief/strategy)

**Status:** ⚙️ SCAFFOLD (architecture proven, real-world validation pending)

**Connects to:** CASCADE (which beliefs change under contradiction), HARMONIA (rhythmic oscillation of learning), MICROORCIM (measurement of convergence quality)

**Real-world:** Cerebellar learning, trading algorithms, organizational feedback loops

---

### 4. MICROORCIM (Sovereignty & Drift Detection)
**What it does:** Measure how much a system is "itself" vs. drifting. Real-time stability score.

**Core metric:**
```
μ = (1 − ρ_drift) · ρ_stability
  where ρ_drift = distance from original goals
        ρ_stability = Lyapunov stability measure (λ < 0)
```

**Status:** ⚙️ SCAFFOLD (formulas ready, live measurement instruments not yet built)

**Connects to:** AURA (detects invariant violations as drift), TRIAD (tracks convergence during learning), EARNED LIGHT (energy budget constraint)

## HOW THEY INTERLOCK

### The Knowledge Reorganization Loop (CASCADE → AURA → TRIAD)
1. **Contradictions pile up** → Truth pressure Π increases (CASCADE)
2. **System approaches threshold** → Invariants begin to strain (AURA)
3. **Reorganization triggered** → TRIAD kicks in (Observe → Correct phase)
4. **New state sought** → Harmonic resonance guides (HARMONIA)
5. **Invariants preserved?** → Checked and maintained (AURA constraint penalty)
6. **Convergence guaranteed?** → Banach fixed-point theorem applies (CHRYSOPOEIA)
7. **New state stable?** → MICROORCIM drift detection confirms

### The Consciousness Constraint Loop (EARNED LIGHT ↔ LAMAGUE ↔ HARMONIA)
1. **Consciousness seeks expression** → State transitions cost energy (EARNED LIGHT)
2. **State described formally** → LAMAGUE grammar constrains possible transitions
3. **States in resonance?** → HARMONIA determines stability and cost
4. **Optimization needed** → Reduce discord (dissonance) → lower energy cost

### The Ethical Constraint Loop (AURA ↔ MICROORCIM ↔ ANAMNESIS)
1. **System operates** → MICROORCIM monitors drift
2. **Drift detected** → AURA invariants invoked
3. **Truth about drift?** → ANAMNESIS confirms via convergence principle
4. **Correction made** → MICROORCIM confirms return to baseline

```
dΨ/dt = k₁·(Π − Π_th)           [CASCADE drive: truth pressure pushes reorganization]
        − k₂·(Ψ − Ψ_inv)         [AURA pull: return to stable minimum]
        − k₃·I_violations        [Invariant penalty: resist violating AURA]
        + k₄·(E_avail/E_need)    [EARNED LIGHT: energy budget allows change]

Where:
  Π = (E·P)/S                    [CASCADE truth pressure formula]
  Ψ ∈ [0, 1]                     [System coherence]
  Ψ_inv ≈ 0.70                   [AURA minimum coherence floor]
  I_violations ∈ {0,...,7}       [AURA invariant count]
  k₁, k₂, k₃, k₄                 [Empirically TBD coupling constants]
```

**What this says:**
- Systems are pushed by truth (CASCADE) and pulled by stability (AURA)
- Invariant violations are costly (AURA enforcement)
- Change requires energy budget (EARNED LIGHT constraint)
- The balance determines when and how reorganization happens
