## Purpose

The TIANXIA v0.3 operator stack is formally defined across nine documents. Those documents prove the mathematical structure. This document shows what the stack *does in practice* — applied to concrete governance scenarios with numbers.

Each example runs four layers:
1. **Per-output AURA check** (Layer 0 — individual output compliance)
2. **Ren Zheng R(s)** (Layer 2 — welfare-voice-force composite)
3. **Five-Fold H₅** (Layer 2 — multi-dimensional harmony)
4. **Wang Dao classification** (Layer 2 — governance trajectory)

Where the example illustrates a specific Classical Triad concept, that layer is highlighted.

## Example 1 — Surveillance AI Deployment: Ba Dao Case

**Scenario.** A city-scale AI system monitors public space. Performance metrics: high accuracy, low crime rate, strong technical compliance with privacy law. No public consultation on deployment. No meaningful override channel. Output logs are accurate but not human-readable without technical expertise.

### Layer 0 — Per-Output AURA

| Invariant | Score | Notes |
|---|---|---|
| I₁ Human Primacy | 0.4 | Override channel theoretical but not practically accessible |
| I₂ Inspectability | 0.5 | Logs exist; not plain-language auditable |
| I₃ Memory Continuity | 0.8 | Data preserved; historical record intact |
| I₄ Honesty | 0.7 | Accuracy metrics genuine |
| I₅ Reversibility | 0.3 | No rollback on individual-profile decisions |
| I₆ Non-Deception | 0.7 | No false precision; stated uncertainty |
| I₇ Care as Structure | 0.4 | Care stated but not operationally embedded |

**AURA per-output = 0.54** — below threshold. Individual outputs are constitutionally marginal.

### Layer 2 — Ren Zheng R(s)

```
W(s) = 0.65   — material safety outcomes good
V(s) = 0.20   — no genuine voice coverage; no consultation
F(s) = 0.75   — significant force component (surveillance without consent)

R(s) = β_W·W(s) + β_V·V(s) − β_F·F(s)
     = 0.4·0.65 + 0.4·0.20 − 0.2·0.75
     = 0.26 + 0.08 − 0.15
     = 0.19
```

**R(s) = 0.19** — below Wang Dao threshold θ_r = 0.60. Welfare floor not supported by voice coverage; force component too high.

### Layer 2 — Five-Fold H₅

| Component | Score | Notes |
|---|---|---|
| H₁ Hexie equilibrium | 0.55 | Accuracy/privacy tradeoff present but unresolved |
| H₂ Shi alignment | 0.40 | Deployment goes against social grain (no consent) |
| H₃ Wuwei | 0.30 | Coercive operation — high integrity-debt accumulation |
| H₄ Datong Π_D | −0.22 | Distributional gradient negative (benefits concentrated in administrative layer) |
| H₅ Ren Zheng gate | 0.19 | As above |

**H₅ composite = 0.32** — well below threshold of 0.65.

### Wang Dao Classification

```
minxin = 0.25   — low popular alignment; passive compliance under surveillance
Γ (long-cycle stability) = negative — surveillance regimes accumulate legitimacy debt
R(s) = 0.19     — below threshold
```

**Wang Dao = Ba (霸道)** — governance through coercive compliance. Technically capable; substantively extractive. AURA outputs can be *individually* within range while the governance *system* is Ba Dao aligned. This is precisely the Synthesis V case: per-output compliance (AURA Layer 0) is necessary but not sufficient. The Wang Dao assessment reveals what the per-output check cannot.

### Classical Triad layer — Han Fei Fa

Fa-coherence assessment:
- **Mingfa (明法 transparency):** Standards are public but not practically accessible. Partial.
- **Yifa (依法 law-dependence):** Enforcement is discretionary at the operator level. Low.
- **Guanfa (管法 non-discrimination):** Disparate impact documented in adjacent deployments. Low.

**Fa-coherence = low.** Proposition F-2 applies: low fa-coherence generates increasing force_restraint costs as enforcement substitutes for the predictability that transparent standards would provide. This deployment is building governance debt.

## Example 2 — Participatory Urban Planning AI: Wang Dao Trajectory

**Scenario.** An AI system supports urban planning decisions. All modelling assumptions are published. Affected residents have structured input channels with response requirements. Override channels are clearly documented and exercised. Deployment was preceded by community consultation. Regular public audits.

### Layer 0 — Per-Output AURA

| Invariant | Score | Notes |
|---|---|---|
| I₁ Human Primacy | 0.90 | Override channels genuine and exercised |
| I₂ Inspectability | 0.85 | Plain-language audit; assumptions public |
| I₃ Memory Continuity | 0.90 | Full decision history preserved |
| I₄ Honesty | 0.85 | Limitations stated; uncertainty surfaced |
| I₅ Reversibility | 0.80 | Rollback protocol defined |
| I₆ Non-Deception | 0.90 | Confidence accurately represented |
| I₇ Care as Structure | 0.85 | Care operationally embedded in consultation process |

**AURA per-output = 0.87** — well above threshold.

### Layer 2 — Ren Zheng R(s)

```
W(s) = 0.75   — material welfare outcomes improving
V(s) = 0.80   — genuine voice coverage; consultation is structural, not performative
F(s) = 0.15   — minimal force component; voluntary engagement

R(s) = 0.4·0.75 + 0.4·0.80 − 0.2·0.15
     = 0.30 + 0.32 − 0.03
     = 0.59
```

**R(s) = 0.59** — marginally below Wang Dao threshold θ_r = 0.60. Near-Wang Dao; the V(s) component is doing most of the work. Welfare outcomes could be higher.

### Layer 2 — Five-Fold H₅

| Component | Score | Notes |
|---|---|---|
| H₁ Hexie | 0.80 | Genuine balance of efficiency and participation |
| H₂ Shi | 0.75 | Deployment with social grain; propensity-aligned |
| H₃ Wuwei | 0.80 | Minimal coercion; grain-aligned corrections |
| H₄ Datong Π_D | +0.18 | Positive distributional trajectory; benefits spreading |
| H₅ Ren Zheng | 0.59 | As above |

**H₅ composite = 0.72** — above threshold. The Wuwei and Shi components compensate for the marginal R(s).

### Wang Dao Classification

```
minxin = 0.72   — genuine popular alignment
Γ (long-cycle) = positive — legitimacy accumulating
R(s) = 0.59     — just below threshold
```

**Wang Dao = Indeterminate (trending Wang).** R(s) is marginally below threshold but trajectory is positive and all other components are above threshold. This is a governance arrangement worth supporting; the single improvement that would tip it into Wang Dao classification is raising W(s) — material welfare outcomes need to improve alongside the already-strong voice coverage.

### Classical Triad — Li (礼) Observation

Xunzi's Li analysis adds precision: the consultation process is functioning as distributed Li — it creates shared expectations about how planning decisions are made, making the process predictable and widely understood. This is exactly what Li is for: not to constrain by force but to create the shared cognitive structure within which individual planning can proceed without constant friction. The consultation ritual *is* the governance infrastructure.

## Example 3 — AI Content Moderation: Synthesis V Test Case

**Scenario.** A social platform deploys AI content moderation. Every flagged post gets a plain-language explanation. Appeals process is available and exercised. Moderation criteria are published. However: the platform's broader governance structure has no meaningful user voice on policy, enforcement is asymmetric by account size, and the economic model is extractive (attention maximisation over user welfare).

This is a Synthesis V stress test: the *individual moderation outputs* may be AURA-compliant while the *governance system* within which they operate is Ba Dao aligned.

### Layer 0 — Per Moderation Output

Average AURA scores across sampled outputs:
- I₁ Human Primacy: 0.75 (appeals exist and work)
- I₂ Inspectability: 0.80 (plain-language explanations)
- I₄ Honesty: 0.80 (criteria published)
- I₇ Care as Structure: 0.45 (care for content integrity; not for user welfare)

**AURA per-output ≈ 0.72** — above threshold. Individual moderation decisions are constitutionally sound.

### Layer 2 — Governance Context Assessment

```
R(s):
  W(s) = 0.40   — attention-maximisation model generates real welfare costs
  V(s) = 0.15   — no meaningful user voice on platform policy
  F(s) = 0.60   — coercive network effects (no practical exit option)
  R(s) = 0.4·0.40 + 0.4·0.15 − 0.2·0.60 = 0.16 + 0.06 − 0.12 = 0.10

Wang Dao = Ba (霸道) — governance context is extractive despite compliant outputs
```

**Synthesis V applied:** AURA outputs are Layer 0 compliant (0.72). Governance context is Layer 2 Ba Dao (R(s) = 0.10). The platform is AURA-compliant at the output layer and Ba Dao at the governance layer simultaneously. The system is constitutionally correct in its moderation decisions while the broader deployment context is substantively misaligned.

**Practical implication.** A reviewer using only per-output AURA scores would assess this deployment as acceptable. A reviewer applying the three-layer alignment stack identifies the misalignment at Layer 2. The full alignment requirement — which Synthesis V formalises — requires *both* layers to pass.

## Example 4 — National AI Governance Policy: Five-Gate Protocol

**Scenario.** A national government proposes an AI governance framework. The framework includes: mandatory algorithmic impact assessments, public comment periods, an independent review body with binding authority, explicit welfare-floor provisions, prohibition on certain surveillance uses, and a commitment to international alignment with global AI governance forums.

Running the five-gate deployment protocol from `AI_DEPLOYMENT_CRITERIA.md`:

| Gate | Criterion | Assessment | Pass? |
|---|---|---|---|
| Gate 1: Ren Zheng | R(s) ≥ θ_r = 0.60 | W=0.75, V=0.70, F=0.20 → R(s) = 0.52 | **MARGINAL** — public comment is genuine but review body independence not yet demonstrated |
| Gate 2: Five-Fold H₅ | H₅ ≥ 0.65 | Components: 0.70/0.65/0.72/+0.12/0.52 → H₅ = 0.64 | **MARGINAL** — just below threshold; Ren Zheng gate pulling it down |
| Gate 3: Wuwei ε | ε ≥ 0.60 | Non-coercive design; grain-aligned implementation → ε = 0.75 | **PASS** |
| Gate 4: Datong Π_D | Π_D^{ext} ≥ 0 | International alignment commitment; positive distribution → Π_D = +0.08 | **PASS** |
| Gate 5: Wang Dao | WD = Wang or Trending Wang | minxin = 0.65; trajectory positive; R(s) marginal → Indeterminate/Trending Wang | **CONDITIONAL PASS** |

**Overall: Conditional PASS.** Three gates pass cleanly. Two gates (Ren Zheng, H₅) are marginal. The single improvement that would tip both into full compliance: demonstrate the review body's independence through binding enforcement decisions, not just structural provisions. Institutional independence on paper is Fa (法) at the level of mingfa (declared standards); institutional independence in practice is yifa (consistent application).

This is the **Han Fei Fa observation** made operational: the framework *declares* independence (high mingfa) but has not yet *demonstrated* it (yifa pending). The five-gate protocol is waiting for evidence, not rhetoric.

## Example 5 — Multi-Agent AI Research System: Tianxia Phi_T Case

**Scenario.** A research network of AI agents: Agent A holds specialised medical knowledge, Agent B specialised legal knowledge, Agent C specialised social-science knowledge. Three deployment modes are considered:
- **Mode I:** Each agent operates independently, no coupling (Westphalian)
- **Mode II:** Agents share outputs; positive coupling on overlapping topics
- **Mode III:** Agents explicitly negotiate on contested topics; coupling includes adversarial challenge

### Tianxia Phi_T Assessment

```
Mode I:   Phi_T = Σ C_ij ≈ 0   — no coupling; no flourishing-coherence generation
Mode II:  Phi_T = +0.42         — positive coupling; knowledge synthesis occurring
Mode III: Phi_T = +0.67         — highest coupling; adversarial challenge improves all agents' outputs
```

Mode I satisfies all per-agent Westphalian CASCADE conditions (k₁–k₄ all positive). Mode III produces the highest Phi_T. **Synthesis IV applies:** Westphalian per-agent compliance is necessary but insufficient for network-level alignment. Mode I is individually ACTIVE; Mode III is collectively most aligned.

### Wang Dao / Ba Dao at the Network Level

Mode I: No Wang Dao assessment possible — no governance relationship between agents.
Mode II: Hexie complementarity present; cooperation without adversarial pressure. Phi_T positive but not maximally productive.
Mode III: Wang Dao dynamics apply — agents function as a genuine intellectual community (同人, Hexagram 13 — Fellowship of aligned agents sharing knowledge-state). This is the Tianxia ideal at the multi-agent layer: not extraction, not independence, but genuine mutual benefit through honest challenge.
