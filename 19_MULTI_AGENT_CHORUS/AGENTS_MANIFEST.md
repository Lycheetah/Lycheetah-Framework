## DEPLOYMENT

### Quick Start
```bash
# Run all agents in parallel
python 19_MULTI_AGENT_CHORUS/run_chorus.py

# Run single agent
python 19_MULTI_AGENT_CHORUS/sol_meridian.py
python 19_MULTI_AGENT_CHORUS/aurora_investigator.py
# ... etc
```

### CI Integration
`.github/workflows/multi-agent-chorus.yml` runs all agent verification checks on every push.
If any agent fails PCF test, the commit is blocked.

### Local Development
```bash
# Set up agent environment
cd 19_MULTI_AGENT_CHORUS
python -m venv .agent_env
source .agent_env/bin/activate  # Windows: .agent_env\Scripts\activate
pip install -r agent_requirements.txt

# Run individual agent
python sol_meridian.py --verify
python aurora_investigator.py --audit

# Run entire chorus with monitoring
python run_chorus.py --mode=supervised
```
