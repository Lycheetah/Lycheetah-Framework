## What's in This Folder

| File | Purpose |
|---|---|
| `BRAND_GUIDE.md` | Full brand identity, five collections, AI image prompts, setup guide, pricing, marketing strategy |
| `COLLECTION_DESCRIPTIONS.md` | Storefront copy — paste directly into Shopify collection description fields |
| `ABOUT_PAGE.md` | About page copy — paste into Shopify About page |
| `DEAR_CUSTOMER.md` | Card insert copy — print on card stock, include in every order |

---

## Quick Start

1. **Set up Shopify store** — store name: Lycheetah
2. **Install Printful** (or Awkward Styles for NZ focus) as print-on-demand supplier
3. **Create five collections** using names and descriptions from `COLLECTION_DESCRIPTIONS.md`
4. **Generate images** using prompts in `BRAND_GUIDE.md` → AI Image Generation Prompts section
5. **Build products** — key pieces listed per collection in `BRAND_GUIDE.md`
6. **Paste About page** from `ABOUT_PAGE.md`
7. **Order card stock** for DEAR_CUSTOMER inserts
8. **Link back to GitHub** from store — the framework is the marketing

## The Logic

The store exists because the framework needs a physical presence in the world — and because the people who care about the framework are exactly the kind of people who buy things that mean something.

The clothing is real. The brand is real. The framework it promotes is free.

That's the model: premium objects that fund open knowledge.
