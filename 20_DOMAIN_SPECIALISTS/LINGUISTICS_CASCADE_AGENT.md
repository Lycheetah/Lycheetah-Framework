## II. DOMAIN PARAMETERS

### What is E (Evidence Strength)?
- External pressure: New contact with other language (Vikings, colonization)
- Internal pressure: Homophones creating ambiguity, case system becoming redundant
- Measurement: Frequency of contradiction in corpus

### What is P (Prior Violation)?
- How much does the new word/form contradict existing grammar?
- Example: English "ain't" — violates standard negation system
- Measurement: Distance from existing patterns

### What is S (Coherence Strain)?
- How integrated is the language system?
- Synthetic languages (high S): German case system, Latin declensions
- Analytic languages (low S): Modern English, Vietnamese
- Measurement: Number of rule-dependency networks

### Critical Π Threshold
- For language: Π ≈ 1.5 (languages change slowly; high S provides resistance)
- Below 1.5: Words borrow, sounds shift, but grammar holds
- Above 1.5: Grammar reorganizes (case systems drop, word order fixes)

## IV. CASCADE SEQUENCES IN LINGUISTICS

### Pattern 1: Contact-Induced Reorganization
Language A + Language B → Language A reorganizes

Example: English + French (Norman Conquest)
- High E: Vocabulary pressure (50% of English now Romance)
- High P: Grammar violation (SVO invading SOV patterns in syntax)
- High S: Old English had complex case system
- Result: Π exceeds threshold → Case system dissolves

### Pattern 2: Internal Pressure
Contradictions within language itself → reorganization

Example: English vowel system
- High E: Sound changes (Great Vowel Shift)
- High P: Vowels become indistinct (phonetic merger)
- High S: Vowel system is load-bearing
- Result: Π builds → Vowel system reorganizes

### Pattern 3: Semantic Merger
Multiple words serve same function → consolidation

Example: "You" replacing "thou"
- High E: Social pressure (you-forms become prestige)
- High P: Violates singular/plural distinction
- High S: Person system is coherent
- Result: Π exceeds → Thou/you distinction collapses
