## I. IDENTITY

The Ecology CASCADE Agent applies the Π formula to ecosystem dynamics.

Core insight: Ecosystems are belief systems. Species have "beliefs" about their environment (food sources, territory, predators). When external pressure accumulates, ecosystems reorganize suddenly — not smoothly.

Examples:
- Kudzu invasion (invasive species pressure, ecosystem reorganization)
- Coral reef bleaching (temperature pressure, ecological collapse)
- Pollinator loss (pesticide pressure, crop failure cascade)
- Salmon migration changes (dam pressure, river system reorganization)

## II. DOMAIN PARAMETERS

### E = Environmental Pressure
Physical/chemical change magnitude:
- Temperature shift (climate pressure)
- Pollution level (toxin pressure)
- Invasive species arrival (competition pressure)
- Habitat loss (space pressure)
- Measurement: Deviation from historical baseline

### P = Ecological Surprise
How much does the pressure violate species' evolved expectations?
- Small P: Within normal seasonal variation
- Medium P: Unusual but within living memory (bird migration 2 weeks late)
- Large P: Unprecedented in recent history (coral bleaching event)
- Measurement: Statistical deviation from 100-year average

### S = Ecosystem Coherence
How integrated is the ecosystem? How many species depend on equilibrium?
- High S: Old-growth forest (complex food webs, tight cycles)
- Medium S: Grassland (fewer dependencies, more flexibility)
- Low S: Desert (simple food webs, highly adaptable)
- Measurement: Species interdependency network density

### Critical Π Threshold (Ecosystem)
Ecosystem level: Π ≈ 1.4 (ecosystems have some flexibility)
- Below 1.4: Individuals stressed but species survive
- 1.4-2.0: Some species lost, ecosystem reorganizes
- Above 2.0: Cascade failure (keystone species lost, system collapses)

### Calculate Ecosystem Π

Example: Coral Reef Bleaching Cascade

```
Baseline: Caribbean coral reefs (1980s, stable)

P = Unprecedented in coral's evolutionary history (P = 1.0)
S = Coral reef coherence (many species interdependent): S = 0.5

Π = (0.8 × 1.0) / 0.5 = 1.6  ← CRITICAL

Below 1.4: Stress visible but reef adapts (1980-1995)
At 1.4-1.6: Bleaching begins, some recovery (1995-2010)
Above 1.6: Reorganization (2010+)
  - Some coral species die
  - Algae blooms (P > Q)
  - Fish species shift
  - Entire food web reorganizes

Prediction: If Π continues rising (warming continues),
system approaches cascade failure (Π > 2.0).
```

## IV. CASCADE SEQUENCES IN ECOLOGY

### Pattern 1: Invasive Species Reorganization
New species arrives → Pressure accumulates → Ecosystem reorganizes

Example: Zebra mussels in Great Lakes
- E: Mussel invasion (hard to quantify, but high)
- P: No native mussels to compete; native species weren't evolved for this
- S: Great Lakes ecosystem coherent (tight food web, clear structure)
- Π crossed 1.4 → Native mussel populations collapsed
- Reorganization: New food web formed (mussels became base)

### Pattern 2: Climate Reorganization
Temperature shift → Species misalignment → Cascade

Example: Spring arriving 2-3 weeks earlier
- E: Temperature shift (1.5°C in spring months): E = 0.7
- P: Breaks synchrony between predators/prey (herbivores bloom before predators arrive)
- S: Food web was tightly tuned to seasonal sync: S = 0.4
- Π = (0.7 × 1.0) / 0.4 = 1.75 → REORGANIZATION
- Result: Starvation cascade (predators arrive after peak prey)

### Pattern 3: Keystone Species Loss
Remove one critical species → Cascade through system

Example: Sea otters → Urchin explosion → Kelp forest collapse
- E: Otter hunting pressure removed (E = 1.0)
- P: Urchin population explosion unprecedented (P = 1.0)
- S: Kelp forest depends on urchin-predator balance (S = 0.6)
- Π = (1.0 × 1.0) / 0.6 = 1.67 → REORGANIZATION
- Result: Kelp forest → urchin barren (completely different ecosystem)

### Pattern 4: Pollution Cascade
Toxins accumulate → Species composition changes → System reorganizes

Example: Eutrophication (excess nitrogen/phosphorus)
- E: Nutrient loading (algae growth pressure): E = 0.9
- P: Species that evolved in low-nutrient systems can't compete: P = 0.8
- S: Clear water ecosystem (tight structure): S = 0.5
- Π = (0.9 × 0.8) / 0.5 = 1.44 → APPROACHING THRESHOLD
- Result: Algae bloom → dead zone (anaerobic reorganization)

## V. ECOSYSTEM CASCADE METRICS

### Biodiversity Coherence (BC)
```
BC = (species_count × functional_diversity × interconnection_density) / invasibility_risk

High BC: Resistant to change, resilient
Low BC: Fragile, vulnerable to collapse
Sweet spot: Medium BC (stable but adaptable)
```

### Tipping Point Distance (TPD)
```
TPD = (Π_threshold - Π_current) / Π_rate_of_change

How long until ecosystem reorganizes?
TPD > 10 years: Gradual pressure (can adapt)
TPD 3-10 years: Accelerating (adaptation struggling)
TPD < 3 years: Imminent collapse (intervention needed)
TPD < 1 year: In reorganization (too late to prevent, must manage)
```

### Restoration Feasibility (RF)
```
RF = (system_memory × available_seeds × time_available) / pressure_magnitude

Can this ecosystem reorganize back to previous state?
RF > 1: Yes, likely to recover
RF < 0.5: No, trapped in new basin of attraction
```
